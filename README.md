## ✨ Что это

Одно приложение, в которое собраны три ранее разрозненных материала по архитектуре мессенджера — **курс, схема и банк вопросов** — и связаны между собой по номеру раздела. Цель — не «прочитать», а **научиться воспроизводить**: рисовать схему у доски, проводить сообщение Алиса → Боб и отвечать на углубляющие вопросы вслух.

> Тренируется производство, а не узнавание: сначала отвечаешь сам — потом сверяешься.

## 🧭 Что внутри

| | Вкладка | Что делает |
|---|---|---|
| 🏠 | **Главная** | Дашборд с единым баллом «готовность к интервью» и тремя столпами прогресса |
| 📖 | **Курс** | 26 разделов + справка по сетям, у каждого — тест на 6 вопросов. Раздел засчитывается при 6/6 |
| 🧩 | **Собери схему** | Восстанавливаешь архитектуру по памяти: 7 уровней от скелета до полной схемы со звонками |
| ⚙️ | **API·БД** | Контракты по памяти: собери запрос (метод + путь) по задаче и восстанови таблицы со столбцами и ключами шардирования |
| 🃏 | **Карточки** | 88 вопросов интервьюера с интервальными повторами (SM-2) и тепловой картой уверенности |
| 🧮 | **Прикидки** | Back-of-envelope задачи с генерацией чисел: выбираешь порядок величины — QPS, шарды, fan-out, трафик — с разбором решения |
| 🎤 | **Прогон** | Репетиция интервью целиком: прикидка → сборка уровней у доски → вопросы вслух → итог и вердикт |
| 🗺️ | **Карта** | Живая схема: 21 узел с пояснениями и 15 пошаговых сценариев — доставка, группы, E2EE, звонки |

<details>
<summary><b>🎯 Как считается «готовность к интервью»</b></summary>

<br>

Единый балл 0–100 — взвешенная смесь трёх направлений:

```
готовность = 35% · знание курса (средний балл по тестам)
           + 30% · сборка схемы (собранных уровней / 7)
           + 35% · карточки (глубина интервальных повторов)
```

Уровни: **Старт → Разогрев → Уверенно → Готова.** Кнопка на Главной всегда подсказывает, что подтянуть в первую очередь.

</details>

<details>
<summary><b>🧩 Игра «Собери схему» — как устроена</b></summary>

<br>

Два этапа на каждом уровне:
1. **Расставь блоки** — компоненты перемешаны, ставишь каждый на своё место по роли-подсказке.
2. **Восстанови связи** — кликаешь два узла, чтобы провести стрелку. Считаются ошибки, лучший результат сохраняется.

Режим **🧠 Чистая доска** добавляет фазу 0: сначала перечисляешь состав уровня по памяти (подсказки скрыты) — как у настоящей доски. Фиксируются время сборки и «чистые прогоны» без ошибок и подсказок (✨).

7 уровней наращиваются: `Скелет → Надёжность → Сессия → Медиа → Масштаб → Мультидевайс → Звонки`.

</details>

## ⚙️ Как это работает

- **Чистая статика** (HTML + CSS + JS) — без сборки, бэкенда и зависимостей.
- Светлая и тёмная тема — 🌙 в шапке, по умолчанию подхватывается системная.
- Весь прогресс (пройденные разделы, собранные уровни, карточки) — в `localStorage` браузера.
- Прогресс можно выгрузить в JSON и перенести в другой браузер — кнопки «Экспорт / Импорт» на Главной.
- Вкладки и разделы курса доступны по прямым ссылкам (`#course/6A`, `#map`…) — F5 возвращает на то же место.
- Работает **онлайн на GitHub Pages** и **локально** — достаточно открыть `Public/trainer.html`.

```bash
git clone https://github.com/chernoyarova/messenger_architecture.git
open messenger_architecture/Public/trainer.html   # или просто открой в браузере
```
